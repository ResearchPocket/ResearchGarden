# Research Garden 🌱

A digital workspace for cultivating and organizing research materials, notes, and insights that works alongside [ResearchPocket](https://github.com/ResearchPocket)

## Features
- 📝 Note taking and organization
- 📚 Saving and adding notes from links
- 🏷️ Tagging and categorization
- 🔄 Version control integration

## Getting Started
1. Clone the repository
2. Install dependencies
3. Configure your settings
4. Start growing your research garden
